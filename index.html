Preprocessors features

  1. Variables
  2. Operations and Color Functions

These features work on both SASS and LESS

Variables


$my_awesome_blue: #001F3F; //SASS
@my_awesome_blue: #001F3F; //LESS

$primary_color: $my_awesome_blue;

h1 {
	color: $primary_color;
}

Variable Types

Variable don't have be just colors


$fonts: Helvetica, Arial, sans-serif;
$images_path: "../img/";
$font-size: 1.5em;
$margin: 20px;
$width: 1000px;

Operations and color functions

Math and colors operations


width: $width / 2 ; // = 500px

Color Functions


$light_blue: lighten($my_awesome_blue, 20%);
$invert_blue: invert($my_awesome_blue, 20%);

MIXINS

Mixins don't return anything but output CSS

Functions like lighten() return values


@mixin border-radius($radius: 10px){
	-webkit-border-radius: $radius;
	   -moz-border-radius: $radius;
	        border-radius: $radius;
}
.button {
	...
	@include border-radius(4px);
}

Will output: 


.button {
	...
	-webkit-border-radius: 4px;
	   -moz-border-radius: 4px;
	        border-radius: 4px;
}

NESTING


.preprocessor {
  .is-awesome {
    ...
    &:hover {
      ...
    }
  }
}
// ======= Output ========

.preprocessor {}
.preprocessor .is-awesome {}
.preprocessor .is-awesome:hover {}

Don't over nest (4 level max)

Imports and minification

Imports now work with no performance costs!

Exciting right?

How?

Just import them and let the preprocessor stitch your CSS together


@import "colors";
@import "typography";
@import "layouts";

It can also output a clear or minified version of your CSS

Misconceptions

"I need ruby or node backend"

SASS is written in Ruby

LESS is written in Javascript

This does NOT mean you need ruby or node on your production server to use them.

You need them on your development machine

Less Runs client-side

Less can run client-side, But NEVER use it in production.

It is OK for quick testing or to complile in-browser if you can't access your dev environment.

It is hard to install

Both Ruby and Node are easy to install on any platform

Mac: Ruby is pre-installed, Node.pkg

Windows: RubyInstaller.exe, Node.exe
